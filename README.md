# 🎓 Campus

A modern campus management platform designed to simplify communication, student management, and administrative workflows. The application provides a secure, responsive, and user-friendly experience for students, agents, and administrators.

## ✨ Features

* 🔐 Secure user authentication
* 👨‍🎓 Student dashboard
* 👨‍💼 Agent dashboard
* 🔔 Real-time notifications
* 📊 Interactive dashboard with analytics
* 👤 User profile management
* 📱 Fully responsive design
* ⚡ Fast and optimized performance
* 🔒 Protected routes and role-based access
* 🎨 Modern and intuitive UI

## 🚀 Getting Started

### Prerequisites

* Node.js (v18 or later)
* npm
* Git

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/campus.git

# Navigate to the project
cd campus

# Install dependencies
npm install

# Start the development server
npm run dev
```

## 🔧 Environment Variables

Create a `.env` file in the project root and add the required environment variables.

```env
VITE_API_URL=your_api_url
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
```
